## Notes

- Each idea file includes a summary, rationale, scope, implementation notes, and success criteria.
- The `High ROI` section is the best place to start if you want the fastest improvement in practical usefulness.
- The `Advanced Modeling` section is better suited for deeper changes to the charging logic.
