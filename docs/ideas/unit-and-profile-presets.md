# Unit and Profile Presets

## Summary
Add one-tap presets for common charging routines and usage patterns.

## Why it fits
The app is already fast to use, and presets would reduce the number of manual adjustments for recurring scenarios.

## Scope
- Add quick actions such as `daily top-up`, `overnight charge`, and `trip prep`
- Optionally connect presets to saved car profiles
- Let users apply a preset and still edit values manually afterward

## Implementation Notes
- Model presets as predefined sets of input values
- Show them above the relevant input sections for quick access
- Keep the first version simple and local-only

## Success Criteria
- Users can load a common scenario with one tap
- Presets reduce repeated slider adjustments
- Applied presets remain editable
